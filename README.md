# Titanic Survival Analysis

An exploratory data analysis of the Titanic passenger dataset to uncover 
what factors determined survival using Python, Pandas and Matplotlib.

## Key Questions Explored
- What was the overall survival rate?
- Did gender affect survival odds?
- Did wealth (passenger class) play a role?
- Were children prioritised in the evacuation?
- Did travelling with family improve survival chances?

## Key Findings
- Only 38% of passengers survived
- Women survived at 74% vs 19% for men
- 1st class passengers survived at 2.5x the rate of 3rd class
- Small families (2–4 members) had the highest survival rates
- A 1st class man had 3x better odds than a 3rd class man, wealth 
  mattered as much as gender
